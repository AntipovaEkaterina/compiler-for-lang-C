# compiler-for-lang-C

C language Compiler on C

Complete features:

- Grammar 
- Lexer 
- Parser
- AST 

To run compiler

- make
- ./main test1.txt

Draw a tree AST

- make graph

Важный комментарий:

Данную программу реализовывала вместе со своей одногруппницей- Березиной Марией, поэтому некоторые части кода будут схожи,
но грамматику и парсер каждый писал для своего языка.
